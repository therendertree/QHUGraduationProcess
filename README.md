## 实习相关

实习是大四第一学期的必修，有10学分，要求至少实习**四个月**。

实习类型有三种：校外企业实习、科学研究实践和校内实训实践。其中科学研究实践只有前三年成绩排名在前40%才可以申请。

### 时间线与材料

大三下学期：找实习岗位。学校会推荐一些实习单位和岗位，也可以自己参加社会招聘。

大四上学期：正式开始实习。负责实习的老师会创建QQ群，后续部分文件会通过此群上传。 实习开始前需要递交文件，一般由一位班干部收齐后统一递交院里审核。审核顺序为：班主任->教管部->学管部->实习负责人->分管领导。

注：企业信息（附件4）介绍要包含“国家企业信用信息公示系统”查询公司注册信息。

大四下学期：参加实习答辩。

### 实习答辩

2025届实习答辩时间在第八学期（大四下学期）第六周的周日，在第六周的周一之前一定要交齐所有的文件材料。

答辩时间为10分钟，包括自我陈述7分钟，答辩评审组提问环节3分钟。学生自我陈述部分建议使用PPT等多媒体方式，重点突出、简洁地陈述实习的主要内容，包括实习单位概况、实习岗位工作内容、实习收获与体会、遇到的问题及解决方法等方面;如有实践成果展示的，应提前准备好相关材料，在答辩现场进行清晰的展示。

## 毕设相关

### 时间线及准备的材料

大三小学期：师生双选阶段，在在毕设管理系统选择老师，上传个人电子签名。建议提前和老师取得联系，交流课题。科研类题目对应毕业论文，非科研类题目对应毕业设计。

**实际毕设时间覆盖的是大四两个学期，因此要求提交的指导记录需从大四上学期的第一周开始。可以实时填写，以免后续补三十多条很麻烦。**

大四上学期：

​	第四周：填写毕设任务书。

​	第七周：准备开题报告。

​	第十一周：提交开题报告。

大四下学期：

​	第二周：填写中期报告。

​	第七周：提交中期报告。

​	第十一周：在这之前要提交论文初稿，系统会自动进行一次查重。（不用额外花钱查重，学校要求是查重率和校内互检率均低于30%，还是很好过的）

​	第十二周：开始专家盲审，给出评分。

​	第十二周的周日：答辩。现场可以实时在系统中查看自己得分。答辩结束，需要填写答辩记录表，而后发送给答辩小组秘书。

​	（如果答辩未通过）第十三周：二次答辩。

​	第十四周：上传毕业论文终稿；刻录相关材料；公布优秀毕业论文。

注：计算机学院优秀毕业论文无需二次答辩。

### 毕设答辩

答辩分组一般在专家盲审之后，答辩日之前公布。按照以往经验来看，不会过早。2024届答辩分组在答辩日当天才公布，2025届答辩分组在答辩日前一天晚上十一点公布。

答辩小组通常由5个老师组成，其中包含四个评委老师现场打分和一个小组秘书计时，小组内不会包含自己的毕设导师。

## 离校前的安排

### 时间线

下述所有时间均为大四下学期。

第八周至第九周：填写毕业生登记表和个人评定表。一定要关注这个，每个班填写时间不统一（原因暂时未知），很有可能不幸的你就会被要求两天内写完，而这两个表内容很多。同时表格填写的内容没有任何参考，其中居然还包括“出身”和“成分”，实在不可思议。

第十周：优秀毕业生申请。记得及时关注，没接到通知可以主动问问院里，有可能是上级出于某种原因忘记下发了。会导致因“被迫”错过时间而无法申请的悲剧orz

第十二周：党团关系转接。

第十四周：

​	填写个人档案接收信息。个人档案很重要，接收地可以提前联系。

​	图书馆停止毕业生借阅服务，借阅图书及时归还。

第十五周：毕业典礼，拿到毕业证和学位证，办理离校手续。恭喜毕业！！！

注1：计算机学院不会集体组织拍摄毕业照，需要几个班协调组织。因此具体拍摄时间取决于各班自己。

注2：毕业之后，申请的edu邮箱无法使用，如果有特殊要求，要和信息中心的老师单独申请。
